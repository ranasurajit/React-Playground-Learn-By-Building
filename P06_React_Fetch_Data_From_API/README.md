# 🚀 React Fetch Data from API

A hands-on React project focused on fetching and displaying data from an external API using modern React practices.

---

## 📌 Project Overview

This project demonstrates:
- Fetching data from an API
- Managing async operations using `useEffect`
- Handling loading states
- Rendering dynamic UI with reusable components
- Structuring a scalable React app

---

## 🧠 Key Concepts Covered

- React Hooks (`useState`, `useEffect`)
- Async/Await for API calls
- Conditional Rendering
- Component Composition
- Tailwind CSS for styling
- Clean state management patterns

---

## ⚙️ Features

- 📡 Fetch posts from API
- ⏳ Loading indicator during API call
- 🧩 Reusable Card component
- 🎨 Responsive UI using Tailwind CSS
- 🧱 Clean component-based architecture

## 🚀 Getting Started

### 1. Clone the repository

```bash
git clone https://github.com/ranasurajit/React-Playground-Learn-By-Building.git
cd React-Playground-Learn-By-Building
```

### 2. Install dependencies

```bash
npm install
```

### 3. Run the app

```bash
npm run dev
```
