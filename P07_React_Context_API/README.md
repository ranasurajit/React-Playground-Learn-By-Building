# React Context API - Counter App

#### This project demonstrates global state management in React using the Context API. It implements a counter application where multiple components share and update state without prop drilling.

## Key Features:

- Global state using React Context API
- Custom hook for cleaner context usage
- Increment / Decrement actions
- Memoized context value using useMemo
- Clean and responsive UI

## Setup Instructions:

```
git clone https://github.com/ranasurajit/React-Playground-Learn-By-Building.git
cd project-folder
npm install
npm run dev
```
